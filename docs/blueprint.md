# **App Name**: VendaZap AI

## Core Features:

- Tag Selection Dropdown: Dropdown menu within the chat interface to select predefined sales tags (e.g., greeting, groups, objections).
- Niche Input Field: Text input field for the user to specify the niche and specific details for the sales message.
- AI Message Generation Tool: Generative AI tool that creates a ready-to-use WhatsApp sales message based on the selected tag and niche input by the user.
- Copy to Clipboard Button: Button to copy the generated sales message to the clipboard for easy pasting into WhatsApp.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to convey trust and efficiency in sales communication.
- Background color: Light grey (#F5F5F5) to provide a clean and neutral backdrop.
- Accent color: A bright green (#90EE90) for CTAs like the copy to clipboard button, indicating successful actions.
- Font: 'Inter' for both headlines and body text. Inter is a sans-serif font known for its clarity and modern appearance, ensuring readability and a professional look.
- Use simple, clear icons for the dropdown menu options and the copy to clipboard button.
- Clean and intuitive layout with a clear separation of the tag selection, niche input, and generated message areas.
- Subtle animation on message generation and copy to clipboard to provide user feedback.
